# KisanAI Design System

## Colors
- **Primary:** #166534 (deep green — agriculture, trust)
- **Primary Light:** #22c55e
- **Primary Dark:** #14532d
- **Secondary:** #92400e (warm earth/amber)
- **Accent:** #c2410c (saffron — CTAs)
- **Background:** #fefdf8 (warm off-white)
- **Surface:** #ffffff
- **Foreground:** #1c1917 (warm dark)

## Typography
- Font: Geist Sans (system fallback)
- Large readable text for low-literacy users
- Minimum body text: 16px
- High contrast for outdoor readability

## Spacing
- 4px base unit
- Mobile-first responsive breakpoints
- Large tap targets: minimum 48px

## Components
- Cards with subtle borders
- Rounded corners (xl/2xl)
- Clean whitespace
- Trust badges and source indicators
- Safety notices for risky content

## Principles
1. Mobile-first
2. Farmer-first (low literacy consideration)
3. Vernacular-ready
4. High contrast for outdoor use
5. Large interactive targets
6. Minimal cognitive load
7. Trust signals everywhere
8. No clutter, no fake metrics
