# KisanAI Roadmap

## Phase 1: MVP (Current)
- Landing page with investor/government pitch
- Farmer application: chat, disease detection, weather, mandi prices, schemes
- AI service layer with mock mode
- API routes with validation
- Compliance pages (privacy, terms)
- Admin dashboard scaffold
- Design system and component library

## Phase 2: Pilot (90 Days)
- FPO/KVK partnership deployments
- Real AI integration (OpenAI/Gemini)
- Hindi language support
- Feedback collection and iteration
- Disease detection accuracy improvement
- Mandi price API integration (AgMarkNet)
- Weather API integration (IMD)

## Phase 3: Growth (6 Months)
- Additional Indian languages (Tamil, Telugu, Kannada, Marathi, Bengali)
- More crop coverage
- Partner marketplace expansion
- WhatsApp bot integration
- Database integration (PostgreSQL)
- User authentication
- Analytics dashboard

## Phase 4: Scale (12 Months)
- Enterprise API for agritech companies
- Advanced analytics and insights
- Insurance/lending integrations
- State government partnerships
- Offline mode for low-connectivity areas
- Voice interface support
